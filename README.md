# Pro Se CaseOps (Android + Zero Cost)

This repository gives you a complete, step-by-step system to run a federal civil case using:
- Android tablet only
- Free tools only
- AI-assisted workflows

## Start here
1. Open `docs/01_FIRST_STEPS.md` and do exactly Step 1 first.
2. Then continue in order.

## What's included
- **Exact setup walkthrough:** `docs/01_FIRST_STEPS.md`
- **Daily/weekly operating checklist:** `docs/02_OPERATING_SYSTEM.md`
- **AI role prompts:** `docs/prompts/AGENT_PROMPTS.md`
- **Court filing templates:** `docs/templates/`
- **OCR scripts for Google Colab (Tesseract):** `scripts/colab_tesseract_ocr.py`
- **Contradiction tracker CSV starter:** `docs/templates/contradiction_tracker.csv`
