# 02 — OPERATING SYSTEM (How to stay proactive)

## Daily flow (every day)
1. Pull latest docket event from your sources.
2. Save file to `01_DOCKET` or `02_ORDERS`.
3. Extract deadlines and add to sheet.
4. If opposition makes factual claims, log in contradiction tracker.
5. Queue document drafting tasks.

## Weekly flow (every Sunday)
1. Review all due dates in next 21 days.
2. Build filing plan with buffer dates.
3. Run evidence gap scan:
   - Which claims still need supporting proof?
4. Run rule scan:
   - Local rule compliance gaps.
5. Prepare one emergency fallback draft for likely motions.

## Contradiction workflow
For each statement by defense counsel:
1. Copy exact statement to tracker.
2. Add source docket number and page.
3. Add your contradicting evidence source.
4. Add objective impact (why it matters).
5. Add requested remedy if needed (e.g., strike, compel, sanctions, clarification).

## Draft workflow
1. Start from template in `docs/templates/`.
2. Add caption + intro.
3. Add facts with citations only.
4. Add authority section.
5. Add relief requested.
6. Add signature + certificate.
7. Run AI Rules Auditor check.
8. Export final PDF to `11_FINAL_PDFS_FOR_FILING`.

## Red-line safety rules
- Never invent facts.
- Never guess citations.
- Never accuse misconduct without clean record citations.
- Never miss deadlines because of formatting; file timely first, refine later if needed.
